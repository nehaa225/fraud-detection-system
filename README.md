# 🛡️ AI-Based Fraud Detection System

## 🚀 Overview
This project detects online fraud using Machine Learning and provides real-time alerts and risk analysis.

## 🔥 Features
- Fraud Detection (TF-IDF + Logistic Regression)
- Risk Analysis (0–100%)
- URL Checker
- OCR Screenshot Detection
- Voice Detection (Audio Upload)
- Email Alerts
- Admin Dashboard
- Awareness Quiz

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- SQLite
- Pytesseract

## ▶️ Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
